Projetos publicados

Empatia – Aplicativo de Autoconhecimento e Escuta Ativa

App iOS e Android

  • Criação do app em Flutter com integração completa ao Firebase (Firestore, Auth, Functions, Crashlytics)
  • Backend com Cloud Functions, incluindo validação de pagamentos via Apple e Google Play
  • Sincronização usando Dart Streams e o método listen, permitindo interações em tempo real entre usuários
  • Autenticação segura com Firebase Auth e persistência contínua de dados no Firestore
  • Interação por swipe com cartões, inspirada no Tinder, permitindo ao usuário deslizar cartões para os lados e para cima, com opção de desfazer a ação e recuperar o último cartão exibido
  • Diário pessoal privado com estrutura de feed, onde o usuário pode salvar cartões selecionados e registrar múltiplos comentários em texto. Entradas são exibidas em ordem cronológica reversa

Contexto do projeto: aplicativo educacional que promove autoconhecimento e escuta ativa entre usuários, através de atividades reflexivas realizadas individualmente e em duplas.

Maia – Plataforma Web para Supervisão Clínica com IA

Node.js Web App

  • Desenvolvimento completo do backend em Node.js com deploy em AWS EC2, Nginx e PM2
  • Integração com OpenAI API para assistente virtual em supervisão de casos clínicos
  • Aprimoramento do assistente via RAG (Retrieval-Augmented Generation), com base em documentos clínicos relevantes
  • Autenticação com Firebase Auth e sessões validadas por ID Token
  • Controle de acesso por plano (gratuito/pro/premium), com limites mensais personalizados
  • Desenvolvimento da interface de chat com layout funcional inspirado no ChatGPT, com navegação por conversas salvas e área central de troca de mensagens
  • Interface responsiva e adaptável, com experiência fluida em dispositivos móveis
  • Persistência de dados estruturada no Firestore

Contexto do projeto: plataforma de apoio a psicólogos, oferecendo reflexões, incentivo à autoavaliação e sugestões técnicas fundamentadas em abordagens como TCC e Terapia dos Esquemas.

Modula – Ecommerce Store

App Frontend em React

  • Projeto desenvolvido com React, Vite e TypeScript
  • Gerenciamento global de estado com Zustand, aplicado ao carrinho, produtos e pedidos
  • Interface modular e responsiva, com layout de vitrine e imagem de destaque
  • Navegação entre telas estruturada como uma árvore de componentes, controlada por índices de estado (sem uso de roteamento externo)

  • Arquitetura de componentes pensada para escalabilidade e fácil manutenção

Contexto do projeto: estudo prático de React, TypeScript e arquitetura frontend, com foco em navegação, controle de estado e responsividade.
